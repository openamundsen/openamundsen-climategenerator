# openAMUNDSEN Climate Generator

Stochastic block bootstrap weather generator for creating future meteorological forcing time series from historical observations.

For details see Appendix A in Strasser, U., Warscher, M., Rottler, E., and Hanzer, F. (2024): "openAMUNDSEN v1.0: an open-source snow-hydrological model for mountain regions", *Geoscientific Model Development*, 17, 6775-6797, [https://doi.org/10.5194/gmd-17-6775-2024](https://doi.org/10.5194/gmd-17-6775-2024).
